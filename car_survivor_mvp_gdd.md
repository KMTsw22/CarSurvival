## 1. 프로젝트 개요

### 한 줄 정의

> **자동차 팬을 위한 모바일 서바이버 로그라이크**
> "내 차를 키우고, 강하게 만들고, 자랑하라"

### 게임 요약

| 항목 | 내용 |
|------|------|
| 장르 | 서바이버 로그라이크 (탕탕특공대 / 뱀파이어 서바이버즈 계열) |
| 플랫폼 | PC STEAM |
| 시점 | 탑다운 2D |
| 아트 스타일 | 2D 카툰 (탕탕특공대 느낌) |
| 런 1회 시간 | 10~15분 |
| 세계관 | **과거 / 현재 / 미래 — 3시대를 넘나드는 드라이브** |
| 핵심 차별점 | 시대별 완전히 다른 적·배경·분위기 + 자동차 커스터마이징·수집 |

### 레퍼런스 게임

| 게임 | 참고 요소 |
|------|----------|
| 탕탕특공대 (Survivor.io) | 전체 게임 구조, 모바일 UX, 장소 진행 방식, 런 시간, 장비 시스템 |
| 뱀파이어 서바이버즈 | 자동 전투, 레벨업 선택, 진화 시스템 |
| Need for Speed | 자동차 팬 감성, 커스터마이징 도파민 |

## 2. 시장 분석 및 경쟁사

### 시장 기회

- 탕탕특공대: 출시 2개월 만에 전 세계 매출 **996억 원**, 한국 MAU 최고 **217만 명**
- 뱀파이어 서바이버즈: Steam 보유자 **500만~1,000만 명**, 압도적 긍정 평가 (98/100)
- **핵심 인사이트**: 모바일에서 자동차 테마 서바이버 로그라이크 = 공백 시장


## 3. 타겟 유저

### 1차 타겟 — 자동차 팬

- 연령대: 20~40대 남성 중심
- 특성: 자동차 유튜브, 커뮤니티 활동, 모델카 수집, 튜닝 관심
- 모바일 게임 습관: 가볍게 즐기되 몰입감 원함
- 핵심 욕구: 소유욕 + 과시욕 + 수집 + 커스터마이징

### 2차 타겟 — 서바이버 장르 유저

- 탕탕특공대 / 뱀파이어 서바이버즈 경험자
- 짧고 반복 가능한 런 구조 선호

### 도파민 트리거 → 게임 장치

| 심리 | 게임 장치 |
|------|----------|
| 소유욕 + 과시욕 | 차고(Garage) 수집 전시 |
| 커스터마이징 욕구 | 도색 / 휠 / 바디킷 |
| 희귀템 수집 스릴 | 장비 뽑기 + 레어 등급 연출 |
| 시대 탐험 욕구 | 과거·현재·미래 완전히 다른 세계 경험 |
| 커뮤니티 소속감 | 차고 공유 + 리더보드 (v2) |

## 4. 세계관 및 스토리

### 세계관 ✅ v0.5 전면 변경

**시간을 달리는 드라이버**

알 수 없는 이유로 차량에 시간 이동 능력이 생겼다. 과거의 공룡 시대, 현재의 도시, 미래의 우주까지 — 어디서든 적이 몰려온다. 살아남는 것이 유일한 목표다.

### 스토리 방침

> 탕탕특공대와 동일하게 **최소화**.
> 로딩 화면 1~2줄이 전부. 유저는 스토리가 아닌 게임플레이 루프 때문에 돌아온다.

**로딩 화면 텍스트 (예시):**
> *"시간의 경계가 무너졌다. 과거든, 현재든, 미래든 — 네 차만 믿어라."*

## 5. 시대 & 맵 시스템 ✅ v0.5 전면 변경

> v0.4의 포스트 아포칼립스 6챕터 구조를 **3시대 × 다수 장소** 구조로 전면 개편.
> 탕탕특공대처럼 장소를 1단계부터 순서대로 클리어해야 다음 장소 오픈.
> **단, 시대(과거/현재/미래) 3개는 처음부터 모두 선택 가능.**

### 시대 선택 구조

```
메인 화면 (시대 선택)
├── 🦕 과거 시대 (처음부터 오픈)
│   ├── 1단계 → 클리어 시 2단계 오픈
│   ├── 2단계
│   ├── 3단계
│   ├── 4단계
│   └── 5단계 (v2~)
│
├── 🏙️ 현재 시대 (처음부터 오픈, MVP 시작점)
│   ├── 1단계: 뉴욕 맨해튼 ← MVP 구현
│   ├── 2단계 (v2~)
│   ├── 3단계 (v2~)
│   ├── 4단계 (v3~)
│   └── 5단계 (v3~)
│
└── 🤖 미래 시대 (처음부터 오픈)
    ├── 1단계 → 클리어 시 2단계 오픈
    ├── 2단계
    ├── 3단계
    ├── 4단계
    └── 5단계 (v2~)
```

### 🏙️ 현재 시대 적

| 적 이름 | 외형 | 특성 | 위험도 | 단계 |

## 8. 게임 루프 설계

### 전체 루프

```
메인 화면 (시대 선택)
  → 장소 선택 (해당 시대 오픈된 단계 중 선택)
  → 차량 선택
  → 런 시작
  → 웨이브 생존 + 레벨업 (카드 3장 선택)
  → 보스 처치 or 체력 0
  → 런 종료 (골드·재료 정산)
  → 차고 (장비 뽑기·합성·업그레이드·커스터마이징)
  → 다음 런 (반복)
```

### 세션 설계

| 항목 | 수치 |
|------|------|
| 런 1회 시간 | 10~15분 |
| 일일 플레이 예상 | 2~4런 |
| 총 세션 | 약 20~40분/일 |
| 보스 등장 | 12~15분 구간 |

## 9. 전투 시스템

> 기본 전투 구조는 v0.4에서 확정. 세계관 변경에 따라 무기/스킬 외형 연출만 시대별로 다르게 적용 가능.

### 9-1. 차량 이동 방식

**이동 방향으로 차가 회전** (탕탕특공대 / 뱀파이어 서바이버즈 동일 방식)

- 조이스틱 방향 = 차량이 바라보는 방향
- 관성 없음 — 즉각 반응 (모바일 터치 최적화)

---

### 9-2. 무기 시스템

#### 무기 4종 (확정)

| 무기 | 발사 방식 | 역할 | 단계 |
|------|---------|------|------|
| **기관총** | 이동 방향 전방 연속 사격 | 기본 딜, 빠른 속사 | MVP |
| **유도 미사일** | 가장 가까운 적 자동 추적 | 강타, 보스 특화 | MVP |
| **화염방사기** | 이동 방향 전방 부채꼴 | 근거리 다수 처리 | MVP |
| **하푼 (작살)** | 이동 방향 전방 단발 관통 | 적 속도 감소 / CC | MVP |

## 10. 장비 시스템 ✅ v0.5 신규

> v0.4의 런 중 파츠 드랍 방식을 탕탕특공대 장비 시스템과 동일하게 전면 변경.
> 런 중에 장비를 줍는 것이 아닌, **차고에서 뽑기·합성·업그레이드**로 강화.

---

### 10-1. 장비 슬롯 4개

| 슬롯 번호 | 슬롯 이름 | 역할 | 탕탕특공대 대응 |
|---------|---------|------|--------------|
| 1 | **무기** | 공격 방식·화력 | 무기 |
| 2 | **엔진** | 이동 속도·공격 속도 | 장갑 |
| 3 | **차체** | 체력·방어력 | 전투복 |
| 4 | **특수장치** | 패시브·유틸 효과 | 목걸이·벨트 |

> v2 이후 **타이어** 슬롯 추가 검토 (5번째 슬롯).

---

### 10-2. 장비 획득 방식

**뽑기 (가챠) + 합성**

```
보석 (유료/무료 재화)
  → 상자 뽑기
  → 장비 획득 (등급 랜덤)

동일 장비 3개
  → 합성
  → 상위 등급 장비 1개

런 클리어 보상
  → 골드 + 업그레이드 재료
  → 장비 업그레이드에 사용
```

---

### 10-3. 장비 등급 체계

| 등급 | 색상 | 합성 조건 |
|------|------|---------|
| 일반 | 흰색 | 기본 상태 |
| 우수 | 초록색 | 동일 장비 3개 합성 |
| 레어 | 파란색 | 동일 장비 3개 합성 |
| 엘리트 | 보라색 | 동일 장비 3개 합성 |
| 에픽 | 노란색 | 엘리트 + 동일 장비 2개 합성 |
| 전설 | 빨간색 | 에픽 + 동일 장비 2개 합성 |

### 10-4. 업그레이드 재료

> 자동차 감성에 맞는 이름으로 적용. 런 클리어·일일 보상·이벤트로 획득.

| 재료 | 획득처 | 용도 |
|------|--------|------|
| 정비 매뉴얼 | 런 클리어, 일일 보상 | 기본 장비 업그레이드 |
| 설계 도면 | 고난이도 런, 이벤트 | 레어 이상 장비 업그레이드 |
| 희귀 부품 | 보스 클리어, 뽑기 | 에픽·전설 장비 업그레이드 |

> 재료 이름·종류는 추후 확정 가능. 현재는 구조만 확정.

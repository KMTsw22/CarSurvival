## 1. 프로젝트 개요

### 한 줄 정의

> **자동차 팬을 위한 모바일 서바이버 로그라이크 게임**  
> "내 차를 키우고, 강하게 만들고, 자랑하라"

### 게임 요약

| 항목 | 내용 |
|------|------|
| 장르 | 서바이버 로그라이크 (뱀파이어 서바이버즈 / 탕탕특공대 계열) |
| 플랫폼 | 모바일 (iOS / Android) — MVP 우선 |
| 시점 | 탑다운 2D |
| 조작 | 한 손 조이스틱 (자동 전투) |
| 핵심 차별점 | 자동차 팬 감성 — 커스터마이징, 수집, 차고 자랑 |
| 목표 출시 | MVP 기준 3~4개월 내 |

### 레퍼런스 게임

| 게임 | 참고 요소 |
|------|----------|
| 뱀파이어 서바이버즈 | 자동 전투, 레벨업 파츠 선택, 웨이브 생존 구조 |
| 탕탕특공대 (Survivor.io) | 모바일 최적화 UX, 수익화 모델, 챕터 구조 |
| Carvival (Steam) | 자동차 테마 서바이버 — 모바일 버전 부재가 공백 |

## 2. 시장 분석 및 경쟁사

### 시장 기회

- 뱀파이어 서바이버즈: Steam 보유자 **500만~1,000만 명**, 압도적 긍정 평가 (98/100)
- 탕탕특공대: 출시 2개월 만에 전 세계 매출 **996억 원**, 한국 MAU 최고 **217만 명** (국내 게임 앱 1위)
- **핵심 인사이트**: 모바일에서 자동차 테마 서바이버 로그라이크는 사실상 공백 시장

### 경쟁사 분석

| 게임 | 플랫폼 | 자동차 테마 | 커스터마이징 | 모바일 |
|------|--------|------------|------------|--------|
| Carvival | PC(Steam)만 | O | O | X |
| Car Survivors (itch.io) | PC만 | O | X | X |
| Driveloop: Survivors | PC(Steam)만 | O | O | X |
| Survivor.io (탕탕특공대) | 모바일 | X | X | O |
| Vampire Survivors | 모바일+PC | X | X | O |

### 결론

> PC에는 자동차 서바이버 게임이 존재하지만 **모바일 버전이 없다**.  
> 모바일 서바이버 강자들은 **자동차 테마가 전혀 없다**.  
> 이 두 가지를 합친 게임이 모바일에 **아직 없다** → 블루오션

## 3. 타겟 유저

### 1차 타겟

**자동차 팬 (Car Enthusiast)**

- 연령대: 20대~40대 남성 중심
- 특성: 자동차 유튜브/커뮤니티 활동, 모델카 수집, 튜닝 관심
- 심리: 소유욕, 과시욕, 수집 욕구, 커스터마이징 욕구
- 모바일 게임 습관: 가볍게 즐기되 몰입감 원함

### 2차 타겟

**모바일 캐주얼 로그라이크 유저**

- 탕탕특공대 / 서바이버류 게임 경험자
- 짧은 런 타임(10~15분)의 몰입형 게임 선호

### 유저 도파민 트리거

| 심리 트리거 | 게임 내 장치 |
|------------|------------|
| 소유욕 + 과시욕 | 차고(Garage) 수집 + 다른 유저 방문 기능 |
| 커스터마이징 욕구 | 도색 / 휠 / 바디킷 변경 |
| 희귀템 수집 스릴 | 레어 파츠 드랍 (언박싱 연출) |
| 업그레이드 쾌감 | 파츠 진화 시 차 외형 실시간 변화 |
| 커뮤니티 소속감 | 차고 SNS 공유, 리더보드 |

## 4. 핵심 컨셉

### 한 문장 피치

> "탕탕특공대 구조 + 자동차 팬의 소유욕·수집욕·과시욕"

### 핵심 경험

1. **런(Run)**: 내 차로 적 차량 떼를 자동으로 박살내며 생존
2. **성장**: 레벨업마다 파츠를 골라 내 차가 강해지고 외형이 바뀌는 쾌감
3. **수집**: 레어 파츠 & 희귀 차량 언락 — 내 차고를 채워나가는 재미
4. **자랑**: 내가 꾸민 차를 다른 유저에게 보여주는 소셜 요소

### 탕탕특공대와의 차별점

| 항목 | 탕탕특공대 | Car Survivor |
|------|-----------|-------------|
| 감정 애착 대상 | 익명 캐릭터 | 내가 직접 꾸민 차 |
| 커스터마이징 | 없음 | 핵심 기능 |
| 소셜 자랑 요소 | 약함 | 차고 공유가 핵심 |
| 타겟 유저 | 일반 캐주얼 | 자동차 팬 (명확한 니치) |
| 외형 진화 | 없음 | 파츠 진화 시 차 실시간 변화 |

## 5. 게임 루프 설계

### 전체 루프 구조

```
차고 화면 → 런 시작 → 레벨업(파츠 선택) → 런 종료 → 메타 성장 → 차고 화면(반복)
```

### 세션 타임

- 런 1회 목표 시간: **10~15분**
- 일일 플레이 예상: 2~4런
- 총 세션: 약 20~40분/일

### 상세 루프

#### 5-1. 차고 화면 (런 전)

- 보유 차량 중 1대 선택
- 파츠 슬롯에 이전 런에서 얻은 파츠 장착
  - 슬롯 구성: 엔진 / 타이어 / 무기 / 특수 (MVP: 엔진 + 무기 2슬롯)
- 커스터마이징 화면 접근 가능 (도색, 휠 등)
- "런 시작" 버튼

#### 5-2. 런 (인게임 핵심)

- 탑다운 2D 맵에서 차량 자동 이동 + 자동 공격
- 플레이어 입력: 조이스틱으로 이동 방향만 제어
- 적 차량이 사방에서 웨이브로 몰려옴
- 적 격파 시 **부품 파편(경험치)** 드랍
- 파편 일정량 축적 → **레벨업 발생**

**자동차 테마 고유 메카닉:**

| 메카닉 | 설명 |
|--------|------|
| 연료 게이지 | 시간 = 연료. 소진되면 런 종료 (긴장감 + 자동차 감성) |
| 드리프트 존 | 특정 구역 통과 시 드리프트 → 일시적 공격력 / 회피율 상승 |
| 충돌 데미지 | 적과 정면 충돌 시 상호 데미지 (차량 전투 현실감) |

**맵 배경 (MVP):**

- 도심 고속도로 (야간)
- 서킷 트랙

#### 5-3. 레벨업 — 파츠 카드 선택

- 레벨업 시 게임 일시 정지
- 파츠 카드 3장 랜덤 제시 → 1개 선택
- 파츠 분류:

| 분류 | 예시 | 효과 |
|------|------|------|
| 엔진 | 터보차저, 슈퍼차저 | 이동 속도, 공격 속도 |
| 무기 | 미사일 런처, EMP 펄스, 오일 슬릭 | 공격 방식 추가 |
| 방어 | 런플랫 타이어, 강화 차체 | 피격 데미지 감소 |
| 특수 | NOS 부스터, 드래프팅 | 고유 능력 |

**파츠 진화 시스템:**

- 동일 파츠 레벨업 or 조합 조건 충족 시 **진화** 발생
- 진화 시 차량 외형 파츠가 실시간으로 교체됨 (시각적 쾌감)
- 예: 터보차저 Lv3 + 슈퍼차저 = **트윈터보 (진화)**

#### 5-4. 런 종료

런 종료 조건:
- 연료(시간) 완전 소진 → 성공적 런 완료
- 차량 체력 0 → 런 실패 (파츠는 일부 보존)

런 결과 화면:
- 획득 파츠 하나씩 애니메이션 공개 (언박싱 연출)
- 레어 파츠 획득 시 특별 연출
- 생존 시간, 격파 수, 최고 기록 배지 표시
- 획득 골드 / 스크랩 정산

#### 5-5. 메타 성장 (차고)

런과 런 사이의 영구 성장:

| 항목 | 내용 |
|------|------|
| 차량 언락 | 골드로 새 차량 잠금해제 |
| 파츠 합성 | 동급 파츠 합성 → 상위 등급 파츠 |
| 커스터마이징 | 도색 / 휠 / 바디킷 변경 (능력치 무관, 순수 외형) |
| 차고 전시 | 내 차량 라인업 전시 및 SNS 공유 |

## 6. MVP 범위 정의

### MVP 포함 (반드시 구현)

| 기능 | 우선순위 | 설명 |
|------|---------|------|
| 탑다운 자동 전투 | P0 | 뱀서 동일 구조, 조이스틱 이동만 |
| 레벨업 파츠 선택 | P0 | 카드 3장 선택 |
| 웨이브 적 스폰 | P0 | 사방에서 몰려오는 적 차량 |
| 차량 3종 | P0 | 스포츠카 / SUV / 트럭 — 능력치 상이 |
| 파츠 10~15종 | P0 | 엔진/무기/방어 각 카테고리 |
| 파츠 진화 3~5종 | P0 | 핵심 도파민 — 외형 변화 포함 |
| 연료 게이지 | P0 | 시간 = 연료 개념 |
| 런 결과 화면 | P0 | 파츠 드랍 + 골드 정산 |
| 차고 화면 | P1 | 차량 선택 + 파츠 장착 |
| 도색 커스터마이징 | P1 | 색상 팔레트로 차체 색 변경 |
| 휠 스킨 3종 | P1 | 스프라이트 스왑 |
| 맵 1종 (도심 야간) | P1 | MVP 단일 맵 |
| 골드 / 스크랩 재화 | P1 | 기본 재화 시스템 |
| 광고 수익화 | P1 | 런 종료 후 보상형 광고 |

### MVP 제외 (이후 버전)

| 기능 | 이유 |
|------|------|
| 실제 차량 브랜드 라이선스 | 비용 과다, MVP 불필요 |
| 멀티플레이 | 복잡도 높음 |
| 3D 그래픽 | 개발 비용 대비 효과 낮음 |
| 차고 방문 / SNS 공유 | v2 소셜 기능 |
| 가챠 시스템 | v2 수익화 고도화 |
| 맵 추가 (서킷 등) | v2 콘텐츠 확장 |
| 클랜 / 커뮤니티 기능 | v3 |

## 7. 차량 시스템

### MVP 차량 3종

| 차량 | 컨셉 | 속도 | 체력 | 특성 |
|------|------|------|------|------|
| 스포츠카 | 공격형 | ★★★★★ | ★★★ | 공격 쿨타임 감소 패시브 |
| SUV | 균형형 | ★★★ | ★★★★ | 기본 방어력 +10% |
| 트럭 | 방어형 | ★★ | ★★★★★ | 충돌 데미지 반사 |

### 차량 잠금해제 구조

```
스포츠카 (시작 차량)
    ↓ 골드 500
SUV (2번째)
    ↓ 골드 1,500
트럭 (3번째)
    ↓ (v2 이후 추가 차량)
슈퍼카 / 클래식카 / 컨셉카 ...
```

- 일반 (흰색)
- 레어 (파란색)
- 에픽 (보라색)

## 9. 커스터마이징 시스템

### MVP 커스터마이징 (3종)

#### 9-1. 도색

- 컬러 팔레트 UI
- 기본 색상 12종 (무료)
- 특수 색상 / 메탈릭 (v2 유료 or 골드 구매)
- 구현 방식: 스프라이트 셰이더 색상 교체 (Unity Shader Graph)

#### 9-2. 휠

- 기본 휠 포함 3종 제공 (MVP)
- 스프라이트 스왑 방식으로 구현
- v2: 5~10종 추가, 골드 or 유료 구매

#### 9-3. 데칼 / 스티커

- 차량 위에 레이어 이미지 합성
- 기본 5종 무료 제공
- v2: 확장 팩

### 커스터마이징 원칙

- **커스터마이징 = 능력치 무관** (순수 외형 변경)
- 단, 파츠 진화 시 **외형 파츠가 자동으로 붙는 효과**는 능력치 연동
- "내가 만든 차"라는 감각이 핵심 — 얼마나 쉽게 내 스타일을 만들 수 있나

## 10. 메타 성장 시스템

### 재화 구조

| 재화 | 획득 방법 | 사용처 |
|------|----------|--------|
| 골드 | 런 완료, 적 격파 보너스 | 차량 언락, 커스터마이징 구매 |
| 스크랩 | 파츠 분해 | 파츠 합성 재료 |
| 다이아 (v2) | 유료 구매, 이벤트 | 프리미엄 커스터마이징, 가챠 |

### 영구 업그레이드 트리 (MVP)

```
골드 투자
├── 차량 언락
│   ├── SUV (골드 500)
│   └── 트럭 (골드 1,500)
├── 차고 슬롯 확장 (파츠 보관함 +5)
└── 기본 능력치 소폭 강화 (탕탕특공대 순찰 시스템과 유사)
    ├── 시작 체력 +5%
    ├── 시작 속도 +3%
    └── 파츠 드랍률 +2%
```

### 파츠 관리

- 보관함: 기본 20슬롯 (골드로 확장)
- 파츠 합성: 동급 3개 → 상위 등급 1개
- 파츠 분해: 파츠 → 스크랩 변환

## 11. 수익화 모델 (초안)

### 탕탕특공대 수익 모델 참고

탕탕특공대는 출시 2개월 만에 996억 원 매출 달성. 수익 모델:
- 게임 내 광고 + 인앱구매 병행
- 유저는 광고 제거보다 **챕터 패키지, 월정액 카드** 선호

### MVP 수익화 전략

#### 광고 (주 수익원 — MVP)

| 광고 유형 | 위치 | 보상 |
|----------|------|------|
| 보상형 광고 | 런 종료 후 | 골드 x2 or 파츠 추가 드랍 |
| 보상형 광고 | 런 실패 시 | 1회 부활 |
| 전면 광고 | 차고 이동 시 (선택적) | 없음 |

#### 인앱구매 (v1 후반~v2)

| 상품 | 가격 (예상) | 내용 |
|------|-----------|------|
| 광고 제거 | 4,900원 | 전면 광고 완전 제거 |
| 스타터 패키지 | 1,900원 | 골드 1,000 + 레어 파츠 3개 |
| 월정액 카드 | 4,900원/월 | 매일 골드 + 스크랩 지급 |
| 프리미엄 커스텀 팩 | 2,900원~ | 특수 도색 / 휠 세트 |

#### 수익화 원칙

- **Pay-to-Win 금지**: 인앱구매가 게임플레이 승패에 직접 영향 없음
- **커스터마이징 유료화**: 능력치 무관 외형 아이템은 유료 판매 가능
- **광고 경험 최적화**: 강제 광고 최소화, 보상형 위주로 유저 자발적 선택

## 12. 기술 스택

### 개발 엔진

**Unity** (2D URP 파이프라인)

선택 이유:
- 모바일 게임 업계 표준
- 풍부한 에셋 스토어 (차량 스프라이트, 파티클 등)
- iOS / Android 동시 빌드
- 뱀파이어 서바이버즈 클론 오픈소스 레퍼런스 다수

### 기술 구성 (MVP)

| 영역 | 기술 |
|------|------|
| 엔진 | Unity 2022 LTS (2D URP) |
| 언어 | C# |
| 물리 | Unity Physics 2D |
| UI | Unity UI Toolkit or UGUI |
| 차량 색상 교체 | Shader Graph (색상 마스크 방식) |
| 로컬 저장 | PlayerPrefs → 추후 클라우드 저장 |
| 광고 | Unity Ads or Google AdMob |
| 분석 | Firebase Analytics |
| 빌드/배포 | Unity Cloud Build |

### 성능 목표

- 타겟 기기: 중급 스마트폰 이상 (iPhone 11 / Galaxy S10 이상)
- 목표 FPS: 60fps (최소 30fps)
- 화면 내 적 최대 동시 스폰: 50~80개

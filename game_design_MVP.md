## 🔄 핵심 게임 루프

### 터널 시스템
- 맵에서 **트로피 3개**를 모으면 → **터널 설치 가능**
- 터널 설치 후 진입 시 → **결계 발동** (이동 가능 공간 한정)
- 결계 안에서 **보스 처치** → 다음 챕터 진입

### ⏱ 10분 타이머 규칙

| 상황 | 결과 |
|------|------|
| 10분 경과 + 터널 **미설치** | ❌ 즉시 게임 오버 |
| 10분 경과 + 터널 **설치 완료** | ✅ 연장 생존 페이즈 돌입 — 터널은 열려 있으며 **플레이어가 원할 때 진입 가능** |

#### 🔥 연장 생존 페이즈 (터널 설치 후)

터널을 설치했다고 해서 바로 들어갈 필요는 없다.  
더 버티면 더 강해지고, 더 강해지면 보스전이 유리해진다. **언제 터널로 뛸지는 플레이어의 판단.**

**몬스터 웨이브 강화 단계 (10분 이후)**

| 단계 | 시점 (예시) | 몬스터 강도 | 보상 |
|------|------------|------------|------|
| 1단계 | 10분 직후 | 🟢 쉬움 강화 | 경험치 ↑ / 골드 ↑ |
| 2단계 | +2~3분 | 🟡 보통 강화 | 경험치 ↑↑ / 보물상자 등장 |
| 3단계 | +4~5분 | 🔴 어려움 강화 | 경험치 ↑↑↑ / 보물상자 ↑ / 골드 ↑↑ |
| 위험 | 이후~ | ☠️ 감당 불가 수준 | — |

> 💡 **플레이어 전략 포인트**  
> 몬스터가 너무 강해져서 감당이 안 된다 싶으면 → **터널로 즉시 런**  
> 조금 더 버틸 수 있다 싶으면 → **더 파밍 후 강화된 상태로 보스 입장**  
> 고위험 고보상 구조 — 욕심 부리다 죽으면 그냥 게임 오버

### 챕터 반복 구조
- 챕터마다 동일한 구조로 반복
- 챕터가 올라갈수록 **트로피 색상 변화** (예: 금색 → 다른 색) 등으로 난이도 시각화

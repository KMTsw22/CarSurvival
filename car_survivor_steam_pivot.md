# Car Survivor — Steam Early Access 전환 정리
> 기반 문서: GDD v0.5 (모바일)
> 작성일: 2026년 3월
> 목적: 모바일 → Steam EA MVP 전환 방향 정리

| 항목 | 모바일 (기존) | Steam EA (변경) |

## 2. 유지 항목 ✅

변경 없이 그대로 가져가는 요소들.

- **코어 게임루프**: 런 → 레벨업 → 카드 선택 → 진화 구조 동일
- **3시대 세계관**: 과거 / 현재 / 미래 구조 유지
- **무기 4종**: 기관총 / 유도 미사일 / 화염방사기 / 하푼
- **스킬 3종 (MVP)**: 오일 슬릭 / EMP 스파크 / 회전 톱날
- **진화 3종 (MVP)**: 화염기관총 / 유도EMP미사일 / 회오리화염
- **차량 3종**: 스포츠카 / SUV / 트럭 (스탯·패시브 동일)
- **차량 커스터마이징**: 도색 / 휠 (능력치 무관 외형만)
- **아트 스타일**: 2D 카툰 탑다운
- **MVP 범위**: 현재 시대 1단계 — 뉴욕 맨해튼
- **적 구성**: 현재 시대 6종 + 보스 1종
- **런 시간**: 10~15분
- **장비 슬롯 4개**: 무기 / 엔진 / 차체 / 특수장치

## 3. 변경 필요 항목 ⚠️

### 3-1. 장비 시스템 (가장 중요)

**기존 (모바일)**: 보석(유료 재화)으로 뽑기 → 동일 장비 3개 합성 → 등급 업

**변경 (Steam)**: 스팀 커뮤니티는 가챠에 강한 거부감. 뱀파이어 서바이버즈 방식 채택.

#### 변경 방향 옵션 (결정 필요)

| 옵션 | 방식 | 장점 | 단점 |
|------|------|------|------|
| A | 런 중 적 처치 시 장비 드랍 | 뱀서 감성, 스팀 친화적 | 밸런스 재설계 필요 |
| B | 런 클리어 후 골드로 상점 구매 | 예측 가능한 성장 | 도파민 약함 |
| C | 런 후 3개 중 1개 선택 (카드 방식) | 탕탕 감성 유지 | 기존 합성 구조 제거 |

> **권장**: 옵션 A + C 혼합 — 런 중 카드 선택(레벨업)에 장비 포함, 런 후 골드로 업그레이드

#### 등급 체계 유지 여부
- 등급 시스템(일반~전설) 자체는 유지 가능
- 단, 등급 상승을 "플레이 횟수·업적"으로 → 유료 뽑기 제거

---

### 3-2. 수익화 모델

**기존 (모바일)**: IAP + 보상형 광고 + 월정액

**변경 (Steam EA)**:

| 단계 | 가격 | 내용 |
|------|------|------|
| Early Access 진입 | $5~7 | 기본 패키지 (할인가) |
| 정식 출시 | $9.99~12.99 | EA 구매자 자동 업그레이드 |
| 선택적 DLC (v2~) | $2.99~4.99 | 추가 시대 콘텐츠 or 코스메틱 팩 |

> **참고 사례**: 뱀파이어 서바이버즈 EA $2.99 → 정식 $4.99 / 탕탕특공대 스팀 버전 무료+IAP (반면교사)

---

### 3-3. 조작 방식

**기존**: 조이스틱 터치 → 이동 방향으로 차 회전

**변경 옵션**:

| 방식 | 설명 | 추천 대상 |
|------|------|---------|
| WASD 이동 + 자동 발사 (이동 방향) | 뱀서 방식 그대로 | 캐주얼, 진입 장벽 낮음 |
| WASD 이동 + 마우스 조준 발사 | 조준 요소 추가 | 좀 더 능동적 전투 |
| 게임패드 완전 지원 | 스틱 이동 + 자동 | 소파 플레이어 |

> **권장**: WASD + 자동 발사(이동 방향) 기본, 게임패드 선택 지원. 마우스 조준은 v2 이후 검토.

## 4. 제거 항목 ❌

스팀 버전에서 완전히 빼는 요소들.

| 항목 | 제거 이유 |
|------|---------|
| 가챠 / 보석 재화 | 스팀 커뮤니티 강한 거부감, 리뷰 폭탄 위험 |
| 보상형 광고 (AdMob) | PC 환경에서 부적절 |
| 강제 광고 | 동일 |
| 연료 게이지 (에너지 제한) | 모바일 과금 유도 장치, PC에서 불필요 |
| 월정액 카드 | 스팀 구독 모델 부적합 |
| 인앱구매 보석 팩 | 패키지 판매로 대체 |

## 5. Steam EA MVP 범위 확정

### 포함 (반드시 구현)

- [ ] 현재 시대 1단계 — 뉴욕 맨해튼 맵
- [ ] 차량 3종 (스포츠카 / SUV / 트럭)
- [ ] 무기 4종 / 스킬 3종 / 진화 3종
- [ ] 현재 시대 적 6종 + 보스 1종
- [ ] 레벨업 카드 시스템 (3장 중 1개 선택)
- [ ] 장비 슬롯 4개 (드랍 방식으로 획득)
- [ ] 장비 등급 시스템 (일반~전설)
- [ ] 골드 기반 장비 업그레이드
- [ ] 차량 커스터마이징 (도색 12색, 휠 3종)
- [ ] 런 결과 화면
- [ ] 차고 화면 (장비 관리)
- [ ] WASD + 게임패드 조작
- [ ] PC UI (1920×1080 기준)
- [ ] Steam 도전과제 기본 5~10개
- [ ] Steam 클라우드 세이브

### 제외 (v2 이후)

- 과거·미래 시대 콘텐츠
- 현재 시대 2단계 이후 (서울, 도쿄 등)
- NOS 부스터 스킬
- 차량 외형 진화 변화
- 멀티플레이 / 리더보드
- 마우스 조준 발사 모드

## 6. Steam 출시 전략 (초안)

- 현재 시대 1단계 완성

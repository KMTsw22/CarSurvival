# 🚗 Car Survivor — 게임 내부 구조

> 게임이 어떻게 돌아가는지 한눈에 보기

---

## 탭 1 — 한 판의 흐름

> 시작부터 터널 통과까지. 10분 안에 일어나는 일들.

| 항목 | 값 |
|------|-----|
| 챕터 제한 시간 | 10분 |
| 수집 아이템 (챕터당) | 3개 |
| 챕터 보스 | 1마리 |

### 순서

**① 맵 시작 — 터널이 보인다**  
맵이 열리는 순간부터 **터널이 저 멀리 보입니다.** 몬스터 무리가 맵을 채우고 있어요. 목표는 단 하나 — 저기까지 가는 것.

↓

**② 달리며 경로 판단**  
미니보스가 지름길을 막고 있습니다. **처치하면 지름길 개방**, 우회하면 더 돌아가야 해요. 수집 아이템이 맵 곳곳에 보입니다 — 어느 경로로 갈지 선택합니다.

↓

**③ 달리며 레벨업 + 수집 아이템 획득**  
몬스터를 박고 드리프트하며 전진합니다. 경험치가 쌓이면 레벨업 — 이동 무기나 패시브를 선택해요. **수집 아이템 3개를 모으면** 터널이 활성화됩니다.

↓

**④ 10분 타이머 압박**  
시간이 지날수록 맵 외곽에서 몬스터가 밀려들어옵니다. 느리게 달릴수록 불리해요. **10분이 지나도 아이템을 못 모으면 게임 오버.**

↓

**⑤ 보스 등장 — 터널 앞을 막는다**  
3번째 아이템을 수집하는 순간 BGM이 바뀌며 **보스가 터널 앞에 등장합니다.** 결계 없어요. 달리던 흐름 그대로 충돌하며 전투 시작.

↓

**⑥ 터널 통과 → 다음 챕터**  
보스 처치 (or 터널 안으로 밀어 넣기) → 터널 개방 → 진입 → **시대 전환 연출** → 다음 챕터 시작.

---

### 연장 페이즈 — 터널을 열고도 더 버틸 수 있다

아이템 3개를 모은 뒤 바로 보스에게 가지 않고 더 파밍할 수 있습니다.  
단, 10분이 지나면 몬스터 밀도가 급증합니다.  
**"언제 보스에게 갈지"** 가 매 판의 핵심 판단이에요.

| 시간 | 변화 |
|------|------|
| 0~3분 | 기본 상태 |
| 3~6분 | 몬스터 밀도 +30% / 새 스폰 포인트 |
| 6~9분 | 외곽부터 안쪽으로 몬스터 밀려들어옴 |
| 9분~ | 경고음 + 밀도 급증 |
| 10분 + 아이템 미수집 | ❌ 게임 오버 |
| 10분 + 아이템 수집 완료 | ✅ 보스 상태 유지 — 계속 진행 가능 |

## 탭 2 — 맵 구조

> 출발 → 중간 → 터널 입구. 3개 레이어 구조입니다.

```
[레이어 1 — 출발 구역]
  · 수집 아이템 1번째 (낮은 리스크)
  · 몬스터 밀도 낮음
  · 기본 조작 익히는 구간
        ↓
  ┌─────────────┬─────────────┐
  │  경로 A     │  경로 B     │
  │  (지름길)   │  (우회로)   │
  │  미니보스가 │  미니보스   │
  │  막고 있음  │  없음       │
  │  처치 시    │  더 돌아감  │
  │  개방+보상  │  몬스터 많음│
  └─────────────┴─────────────┘
        ↓
[레이어 2 — 중간 구역]
  · 수집 아이템 2번째 (미니보스 근처)
  · 몬스터 밀도 중간~높음
  · 신호등 타워 등장
        ↓
[레이어 3 — 터널 입구 근처]
  · 수집 아이템 3번째 (가장 위험)
  · 여기서 수집하면 즉시 보스 활성화
        ↓
[터널 입구]
  · 보스 대기 중
  · 결계 없이 달리던 흐름 그대로 전투
```

### 핵심 설계 원칙

**터널은 항상 보인다**  
맵 어디서든 터널이 화면에 보입니다. "저기까지 가면 된다"는 목표가 항상 명확해요.

**시간이 갈수록 포위된다**  
6분 이후 맵 외곽부터 몬스터가 밀려들어옵니다. 느리게 달릴수록 공간이 줄어들어요.

### 미니보스 배치 원칙

각 챕터마다 미니보스 3마리가 맵에 고정 배치됩니다. 처치는 선택이에요.

- **처치하면** — 지름길 개방 + 골드·파츠 드랍 + 주변 몬스터 일시 감소
- **우회하면** — 체력 유지 + 시간 절약, 단 더 긴 경로

## 탭 3 — 전투 방식

> 무기가 자동으로 쏘는 게 아닙니다. 운전 행동 자체가 전투입니다.

### 기본 조작

#### 박기 — 주 공격
- **발동**: 속도 50 이상 충돌
- **효과**: 빠를수록 데미지가 높아집니다. 저속에서 박히면 반대로 플레이어가 피격
- **연속 박기 콤보**: 0.5초 내 연속 충돌 시 배율 상승

#### 드리프트 — 생존 기술 + 공격
- **발동**: Shift + 방향키
- **효과**: 측면 넉백 발생 + 드리프트 중 피격 30% 감소
- **전략**: 포위당했을 때 탈출하는 핵심 기술. 드리프트 중 이동 무기들이 강화됨

#### 부스트 — 궁극 이동기
- **발동**: E키 (고속 유지 시 충전)
- **효과**: 0.8초 무적 + 고속 돌진
- **용도**: 포위 탈출 또는 보스에게 돌격

---

### 이동 무기 4종 — 운전 행동으로 발동

> 운전 행동 없이는 아무것도 하지 않습니다.

#### 🔥 드리프트 화염 지대
- **발동**: 드리프트 중 (속도 80 이상)
- **효과**: 타이어 마찰열로 바닥에 화염 지대 생성. 2~3초 지속
- **시너지**: 오일 슬릭과 접촉 시 폭발
- **진화**: 드리프트 화염 Lv5 + 오일 슬릭 → **인페르노 트레일**

#### ⚡ 급회전 충격파
- **발동**: 0.3초 내 90도 이상 급회전
- **효과**: 차 주변 360도 넉백 + 짧은 무적. 쿨다운 3초
- **전략**: 포위 탈출 핵심 기술
- **진화**: 충격파 Lv5 + EMP 펄스 → **전자기 폭풍**

#### 💥 충돌 충격탄
- **발동**: 속도 100 이상 충돌 시
- **효과**: 충돌 지점 중심 폭발. 반경 내 추가 광역 데미지
- **진화**: 충격탄 Lv5 + 미사일 포드 → **폭주 탄두**

#### 🌀 테일 슬라이드
- **발동**: 전진 중 급브레이크 + 방향 입력
- **효과**: 차 후미 쓸기 공격. 처치 시 경험치 1.5배
- **전략**: 뒤따르는 적 무리 처리에 최적
- **진화**: 테일 슬라이드 Lv5 + 회전톱날 → **분쇄 선회**

---

### 보조 무기 4종 — 이동 무기와 조합할 때 강해진다

| 무기 | 단독 효과 | 이동 무기 시너지 |
|------|---------|--------------|
| 오일 슬릭 | 차 뒤에서 오일 분사, 적 50% 감속 | 드리프트 화염 지대와 접촉 시 폭발 (진화 재료) |
| EMP 펄스 | 5초마다 주변 전자기 충격 | 급회전 충격파와 동시 발동 시 스턴 범위 2배 (진화 재료) |
| 회전톱날 | 차 주위 톱날 회전 | 드리프트 중 궤도 반경 2배 / 테일 슬라이드 진화 재료 |
| 미사일 포드 | 주기적 유도 미사일 | 고속 이동 중 발사 속도 +50% / 충격탄 진화 재료 |

### 진화 조합 정리

| 이동 무기 (Lv5) | + 보조 무기 | → 진화 무기 |
|---------------|-----------|-----------|
| 드리프트 화염 지대 | 오일 슬릭 | 인페르노 트레일 |
| 급회전 충격파 | EMP 펄스 | 전자기 폭풍 |
| 충돌 충격탄 | 미사일 포드 | 폭주 탄두 |
| 테일 슬라이드 | 회전톱날 | 분쇄 선회 |

### 패시브 7종

| 패시브 | 효과 |
|--------|------|
| SpeedUp | 이동속도 증가 |
| DamageUp | 충돌 및 무기 데미지 증가 |
| HpUp | 최대 체력 증가 |
| HP Regen | 체력 재생 |
| EXP Boost | 경험치 획득량 증가 |
| Damage Shield | 데미지 흡수 방어막 |
| Magnet | 아이템 흡수 범위 증가 |

## 탭 4 — 성장 구조

> 판 안에서 성장하는 인게임 레이어, 판 밖에서 영구 성장하는 메타 레이어.

### 인게임 성장 — 판 안에서, 끝나면 리셋

```
몬스터 처치 → 경험치 → 레벨업 → 무기 / 패시브 선택
```

- 이동 무기 4종 + 보조 무기 4종 + 패시브 7종 중 선택
- 이동 무기 Lv5 + 해당 보조 무기 보유 시 진화 해금
- 공구상자 (신호등 타워 처치 드랍) 로 즉시 강화 가능

### 메타 성장 — 판 밖에서 골드로 구매, 영구 적용

```
게임 종료 → 골드 정산 → 차고(샵) → 영구 패시브 구매 → 다음 판 적용
```

#### A. 수치 강화형

| 업그레이드 | 최대 레벨 | 레벨당 효과 |
|----------|---------|----------|
| 기본 이속 증가 | 5 | +2% |
| 드리프트 지속시간 | 5 | +0.15초 |
| 충돌 데미지 | 5 | +5% |
| 부스트 쿨다운 감소 | 3 | -0.5초 |
| 기본 체력 증가 | 5 | +10% |
| 골드 획득량 | 5 | +8% |
| 경험치 흡수 범위 | 5 | +15% |

#### B. 선택지 확장형

| 업그레이드 | 비용 | 효과 |
|----------|------|------|
| 시대 선택권 | 골드 1,200 | 2챕터(현재)부터 시작 선택 가능 |
| 패턴 스캐너 | 골드 1,500 | 보스 전투 전 1페이즈 패턴 3초 미리 보기 |
| 지도 기억 | 골드 1,000 | 다음 판 시작 시 미니보스 위치 맵에 미리 표시 |
| 드라이브 마스터리 | 골드 1,800 | 드리프트 화염 지대 처음부터 최대 레벨 시작 |
| 타임 리와인드 | 골드 2,500 | 판당 1회 — 게임 오버 시 30초 전 상태 복구 |

### 골드 획득 방법

| 상황 | 획득량 (기준값, 미확정) |
|------|---------------------|
| 일반 몬스터 처치 | 1~3골드 |
| 미니보스 처치 | 50~100골드 |
| 챕터 클리어 | 200골드 |
| No Damage 보스 처치 | 추가 300골드 |
| 연속 처치 콤보 보너스 | 최대 +50% |

## 탭 5 — 3개 챕터

> 공룡 시대에서 시작해 미래 폐허로. 터널이 시대를 연결합니다.

---

### 1챕터 — 공룡 시대 (백악기)

| 항목 | 내용 |
|------|------|
| 배경 | 정글 · 화산 지대. 공룡이 달려든다 |
| 수집 아이템 | 공룡 알 3개 |
| 환경 기믹 | 없음 (튜토리얼 챕터) |
| 색감 | 진한 초록 · 갈색 · 화산 빨강 |
| BGM | 원시적 타악기 + 긴장감 |

#### 챕터 미니보스

| 미니보스 | 경로 | 처치 조건 |
|---------|------|---------|
| 랩터 무리장 | 정글 지름길 차단 | 드리프트 화염 지대로만 접근 |
| 갑옷 트리케라 | 공룡 알 2 근처 차단 | 급회전 충격파 3회 |
| 화산 익룡 | 터널 전 최단 경로 차단 | 하강 타이밍에 고속 정면 충돌 |

#### 챕터 보스 — 렉스 오버로드

> "이 땅 위를 달리는 것은 나뿐이다."

- **외형**: 거대한 T-Rex. 발걸음마다 지면이 진동
- **핵심 운전 조건**: 측면 공격만 유효 — 정면 박기 시 튕겨남
- **환경 연동**: 발걸음마다 맵에 균열 생성. 균열 위 통과 시 일시 감속

| 페이즈 | HP | 패턴 | 운전 요구 |
|--------|-----|------|---------|
| 1페이즈 | 100~66% | 정면 돌진 + 포효 충격파 | 포효 직전 드리프트 후 측면 충돌 |
| 2페이즈 | 66~33% | 랩터 떼 소환 + 꼬리 휩쓸기 | 랩터 드리프트 처리하며 렉스 측면 유지 |
| 3페이즈 | 33~0% | 격노 — 전속력 추격 | 화염 지대 깔아둔 뒤 유인 후 역충돌 |

- **환경 연동**: 화산재 연기 기믹 최대치 활성화

#### 챕터 보스 — 무결 AI 제로스

> "너의 패턴은 이미 분석됐다. 다음 행동을 예측한다."

- **외형**: 홀로그램으로 이루어진 데이터 덩어리. 공격받으면 재조합
- **핵심 운전 조건**: 같은 경로 두 번 사용 금지 — AI가 예측해 차단
- **환경 연동**: 노이즈 구역이 제로스 주변에 확산. 이동 가능 공간 점점 감소

| 페이즈 | HP | 패턴 | 운전 요구 |
|--------|-----|------|---------|
| 1페이즈 | 100~66% | 경로 학습 — 이동 패턴 분석 후 차단 | 매번 다른 경로 사용 |
| 2페이즈 | 66~33% | 미러 유닛 생성 (학습한 경로로 움직임) | 자신의 경로로 복사본 충돌 유도 |
| 3페이즈 | 33~0% | 맵 대부분 노이즈화 | 남은 정상 구역만 통해 드리프트 접근 |
